Consultas y Ejemplos
  • Diferencias entre SQL y NoSQL
    • SQL (Structured Query Language) es un lenguaje de consulta utilizado para manejar bases de datos relacionales.
    • NoSQL (Not Only SQL) es una amplia categoría de sistemas de gestión de bases de datos que no siguen el modelo relacional tradicional.
    • La principal diferencia radica en la estructura de los datos, el enfoque de escalabilidad y flexibilidad.
  • Diferencias entre MySQL y NoSQL
    • MySQL es un sistema de gestión de bases de datos relacional (RDBMS), mientras que NoSQL se refiere a una amplia categoría de sistemas de gestión de bases de datos que no siguen el modelo relacional tradicional.
    • La principal diferencia entre MySQL y NoSQL radica en su enfoque de almacenamiento y gestión de datos.
  • VIEWS en MySQL
    • Las VIEWS son consultas almacenadas que pueden ser tratadas como tablas virtuales dentro de MySQL.
    • Ejemplos de VIEWS en MySQL
      • Crear una vista que muestre solo los productos con un precio superior a cierto valor.
      • Crear una vista que muestre la cantidad total de ventas por cliente.
      • Crear una vista que combine datos de múltiples tablas para mostrar información detallada del pedido.
      • Crear una vista que muestre solo los empleados que pertenecen a un departamento específico.
      • Crear una vista que muestre los productos más vendidos.
  • Procedimientos Almacenados en MySQL
    • Los procedimientos almacenados son un conjunto de comandos SQL que se guardan en el servidor de la base de datos y se pueden ejecutar de manera repetida.
    • Ejemplos de Procedimientos Almacenados en MySQL
      • Crear un procedimiento almacenado para calcular el salario total de un empleado.
      • Crear un procedimiento almacenado para agregar un nuevo cliente a la base de datos.
      • Crear un procedimiento almacenado para actualizar el precio de un producto.
      • Crear un procedimiento almacenado para eliminar un registro de la tabla de pedidos.
      • Crear un procedimiento almacenado para calcular el impuesto sobre las ventas para una factura.
